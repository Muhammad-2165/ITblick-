IT blick

Язык программирования - это система, имеющая набор команд, правил, на основе которых разрабатывают программы.

На этом сайте можно найти определения популярных языков программирования , а также HTML и CSS (HTML И CSS НЕ ЯЗЫКИ ПРОГРАММИРОВАНИЯ!)

Вперёд за знаниями!

PYTHON 

Python - это достаточно легкий и универсальный язык программирования. 
Python очень популярный язык программирования, его можно применить в 
веб разработке, в разработке игр и других программ.

Пример кода:
print("Hello, World!")
JAVASCRIPT 
        	  
JavaScript - это простой язык программирования который примеяют к HTML страничке.
JavaScript язык для веб разработки, но есть технология NodeJs которая адаптирует
язык javascript для  других задач. 

Пример кода:
console.log('Hello, World!')
HTML 

HTML - это простой язык разметки. С помощью 
знаний по HTML можно разработать каркас сайта (тексты, изображения,
таблицы и тому подобное).

Пример кода:
<p> СТРОЧКА </p>
CSS 
 
CSS - это простой язык стилей. С помощью CSS можно стилизировать HTML страничку.
CSS как и HTML тоже составляет каркас сайта.

Пример кода:
div {
      text-align: center;
    }
RUBY 

Ruby - это простой язык программирования, с элегантным синтаксисом.
Ruby универсален , основное
предназначение ruby создание простых программ. 

Пример кода:
puts 'Hello, World!'
JAVA 

Java - это строго типизированный объектно-ориентированный 
универсальный язык программирования. У java простой синтаксис.

Пример кода:
package hello;

public class App {

    public static void main(String[] args) {
        System.out.println("Hello world!");
    }

}
C++ 

C++ - непростой язык программирования для начинающих, но быстрый. С++ универсален.
На С++ можно писать сложные программы. Также c++ используют и для хакинга.

Пример кода:
#include <p>iostream</p>
using namespace std;
int main() {
    cout << "Hello world";
    return 0;
}
C# 

C# - непростой язык программирования для начинающих, но быстрый. С# универсален.
Основное предназначение С# написание игр.

Пример кода:
Console.WriteLine("Hello World!");
C 

C - непростой язык программирования для начинающих, но быстрый. C универсален.
Основное предназначение C написание программ.

Пример кода:

#include lt;p>stdio.hlt;/p>

int main() {
    printf("Hello, World!\n");
    return 0;
}
RUST 
Rust - это непростой язык программирования в целом, но быстрый. 
Rust универсальный язык программирования, который хорошо подходит для
написания сложных систем.

Пример кода:
fn main() {
    println!("Hello, World!");
}
GO 

Go - это многопоточный язык программирования, от известной компании Google.
Go - универсален, сильная сторона Go - веб и многопоточные программы.

Пример кода:
package main

func main() {
    println("Hello, World!")
}
